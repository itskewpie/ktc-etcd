== Description
 Group Definitions 

== Example Group
 {
  "id": "users",
  "gid": "100",
  "append": false
} 

where:
id  =  Group Name
gid =  Group nemeric id 
append =  Set this to true if you want the users that are returned on search to be the exclusive list of users for this group. aka True=add these members to the group, false=these are the definitive group members

== USAGE
Groups have to be defined here so users can successfully subscribe to them. If not you may get an error.
